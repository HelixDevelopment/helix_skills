# I18n-Client-TS

- **GitHub URL**: <https://github.com/vasic-digital/I18n-Client-TS>
- **Description**: Generic client-side i18n wrapper around i18next
- **Category**: TypeScript / React + Web
- **Status**: Active

## Overview

A client-side internationalization wrapper built on i18next for TypeScript and React applications. Provides a simplified API for loading translations, switching locales, and formatting locale-aware content in the browser.

## Tech Stack

- Language: TypeScript
- Framework: React
- Built on: i18next

## Key Features

- i18next wrapper with simplified API
- Locale switching and translation loading
- React hooks for localized content
