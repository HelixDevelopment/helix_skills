## Overview

A type-safe TypeScript client library for the Catalogizer API. Built on axios with built-in authentication, automatic retry logic, and per-domain service modules for organizing API endpoints by feature area.

## Tech Stack

- Language: TypeScript
- HTTP client: Axios
- Target: Catalogizer API

## Key Features

- Type-safe Catalogizer API bindings
- Axios-based HTTP client with retry logic
- Per-domain service organization
