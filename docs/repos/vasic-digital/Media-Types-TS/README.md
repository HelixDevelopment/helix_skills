## Overview

A shared TypeScript type definitions package for the Catalogizer ecosystem. Provides consistent types for media entities, authentication payloads, and API request/response shapes used across all Catalogizer frontend modules.

## Tech Stack

- Language: TypeScript
- Package type: Type definitions library
- Target: Catalogizer ecosystem

## Key Features

- Media entity type definitions
- Authentication payload types
- API request/response type contracts
