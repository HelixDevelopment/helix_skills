## Overview

A generic internationalization (i18n) module for Go applications. Provides locale management, message translation, and pluralization support for building multilingual software.

## Tech Stack

- Language: Go
- Module path: digital.vasic.i18n

## Key Features

- Locale-aware message translation
- Pluralization and gender-aware formatting
- Reusable internationalization primitives
