- **Category**: AI / LLM Provider + Agent

## Overview

A reusable Go module for generating, storing, and querying text and code embeddings. Provides a provider-agnostic interface for embedding generation with support for multiple backends, used as a building block across RAG and search pipelines.

## Tech Stack

- Language: Go
- Module: digital.vasic.embeddings

## Key Features

- Provider-agnostic embedding generation interface
- Batch embedding support for large document sets
- Caching and deduplication of embedding vectors
