- **Category**: TypeScript / React + Web
- **Status**: Active

## Overview

React components for browsing media entities within the Catalogizer application. Provides grid and list views, filtering, search, and navigation through media hierarchies and categories.

## Tech Stack

- Language: TypeScript
- Framework: React
- Target: Catalogizer

## Key Features

- Grid and list view media browsing
- Filtering and search across media entities
- Hierarchical media navigation
