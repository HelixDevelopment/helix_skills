# mdns

- **GitHub URL**: <https://github.com/vasic-digital/mdns>

- **Category**: Messaging + Observability + Storage
- **Status**: Active

## Overview

A Go module implementing RFC 6762 (mDNS) and RFC 6763 (DNS-SD) for multicast DNS service announcement and discovery on local networks. Provides drop-in LAN service registration without centralized DNS infrastructure.

## Tech Stack

- Language: Go
- Module path: digital.vasic.mdns
- Protocols: RFC 6762 (mDNS), RFC 6763 (DNS-SD)

## Key Features

- RFC-compliant mDNS service announcement
- DNS-SD service discovery on local networks
- Drop-in LAN service registration
