- **Category**: Auth + Security + Middleware

## Overview

LeakHub collects and curates prompt-leak examples for red-team training and defensive testing. It provides a corpus of known prompt-extraction attacks and their variants, enabling security teams to train detection models and validate that LLM applications resist adversarial prompt manipulation. Fixtures are labeled for defensive use only.

## Tech Stack

- Language: Multiple
- Framework: Corpus management and fixture generation

## Key Features

- Curated corpus of prompt-leak attack examples
- Red-team training fixtures for adversarial testing
- Labeled variants for detection model training
- Defensive-use governance to prevent misuse
