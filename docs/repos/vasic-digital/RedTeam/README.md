- **GitHub URL**: <https://github.com/vasic-digital/RedTeam>

- **Category**: AI / LLM Provider + Agent
- **Status**: Active

## Overview

A YAML-driven test harness for adversarial prompt regression testing against LLM guardrails. Defines adversarial fixtures in declarative YAML, runs them against guardrail implementations, and reports pass/fail results for continuous security validation.

## Tech Stack

- Language: Multiple
- Config: YAML-driven fixtures

## Key Features

- Declarative YAML adversarial prompt fixtures
- Guardrail regression test execution and reporting
- CI/CD integration for continuous security validation
