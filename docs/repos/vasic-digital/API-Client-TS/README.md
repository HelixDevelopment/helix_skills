# API-Client-TS

- **GitHub URL**: <https://github.com/vasic-digital/API-Client-TS>
- **Description**: Generic REST API client for TypeScript
- **Category**: TypeScript / React + Web
- **Status**: Active

## Overview

A generic, reusable REST API client library for TypeScript applications. Provides a type-safe HTTP client with configurable request/response handling, authentication, and error management.

## Tech Stack

- Language: TypeScript
- Protocol: REST / HTTP

## Key Features

- Type-safe REST API request construction
- Configurable authentication and interceptors
- Generic error handling and retry logic
