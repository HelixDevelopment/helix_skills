# Media-Player-React

- **GitHub URL**: <https://github.com/vasic-digital/Media-Player-React>
- **Description**: React media player component for Catalogizer entity playback
- **Category**: TypeScript / React + Web
- **Status**: Active

## Overview

A React media player component for playing back media entities within the Catalogizer application. Supports audio and video playback with standard player controls and integration with the Catalogizer media library.

## Tech Stack

- Language: TypeScript
- Framework: React
- Target: Catalogizer

## Key Features

- Audio and video playback controls
- Integration with Catalogizer media library
- Standard player transport controls
