## Overview

An Android 15 AOSP build targeting the Orange Pi 5 Max single-board computer. Provides a customized Android system image for the ATMOSphere hardware platform running on ARM64 architecture.

## Tech Stack

- Language: Multiple (Java, Kotlin, C/C++)
- Platform: Android 15 (AOSP)
- Hardware: Orange Pi 5 Max (ARM64)

## Key Features

- Android 15 AOSP build for Orange Pi 5 Max
- Board-specific kernel and driver integration
- ATMOSphere system customization
