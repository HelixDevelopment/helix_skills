# UI-Components-KMP

- **GitHub URL**: <https://github.com/vasic-digital/UI-Components-KMP>
- **Description**: Kotlin Multiplatform UI components: theme system, animations, accessibility utilities for Compose
- **Category**: Cross-platform (KMP) Modules
- **Status**: Active

## Overview

UI-Components-KMP provides shared UI components for Kotlin Multiplatform applications built with Compose Multiplatform. It includes a theme system for consistent visual styling, animation utilities for smooth transitions, and accessibility helpers to ensure UI components meet accessibility standards across Android, iOS, desktop, and web targets.

## Tech Stack

- Language: Kotlin (KMP)
- Framework: Compose Multiplatform, Kotlin Multiplatform

## Key Features

- Cross-platform theme system for consistent visual styling
- Animation utilities for smooth UI transitions
- Accessibility helpers for platform accessibility standards
- Shared Compose UI components across all KMP targets
