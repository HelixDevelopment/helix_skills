## Overview

A comprehensive automated testing tool that captures UI interactions and screenshots across web, desktop, and mobile platforms. Provides a unified interface for recording test sessions, capturing visual state, and generating test artifacts for visual regression and QA workflows.

## Tech Stack

- Language: Multiple
- Platforms: Web, Desktop, Mobile

## Key Features

- Automated UI recording across web, desktop, and mobile
- Screenshot capture with platform-specific adapters
- Test session management and artifact generation
