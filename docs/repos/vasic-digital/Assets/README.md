# Assets

- **GitHub URL**: <https://github.com/vasic-digital/Assets>
- **Description**: Generic, reusable Go module for lazy asset loading with strategy-based resolution
- **Category**: Apps / Tools / Experimental
- **Status**: Active

## Overview

A Go module providing lazy asset loading with pluggable resolution strategies. Enables deferred loading of resources (files, images, configs) with configurable fallback chains and caching behavior.

## Tech Stack

- Language: Go
- Module path: digital.vasic.assets

## Key Features

- Lazy asset loading with deferred initialization
- Strategy-based asset resolution
- Configurable fallback chains
