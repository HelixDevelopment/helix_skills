# WebSocket-Client-TS

- **GitHub URL**: <https://github.com/vasic-digital/WebSocket-Client-TS>
- **Description**: Reusable TypeScript/React module
- **Category**: TypeScript / React + Web
- **Status**: Active

## Overview

A reusable TypeScript WebSocket client library providing connection management, automatic reconnection, message serialization, and React hooks for real-time bidirectional communication in web applications.

## Tech Stack

- Language: TypeScript
- Framework: React (hooks)
- Protocol: WebSocket

## Key Features

- WebSocket connection lifecycle management
- Automatic reconnection with backoff
- React hooks for real-time data streams
