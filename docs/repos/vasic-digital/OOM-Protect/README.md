- **Category**: Apps / Tools / Experimental
- **Status**: Active

## Overview

A Linux system hardening tool that protects critical processes from the Out-Of-Memory (OOM) killer on systemd-based distributions. Configures OOM score adjustments and cgroup protections to prevent essential services from being terminated under memory pressure.

## Tech Stack

- Language: Multiple (shell/scripts)
- Platform: systemd Linux
- Use case: OOM protection

## Key Features

- OOM killer protection for critical processes
- systemd service integration
- Memory pressure management
