## Overview

A reusable Go module providing storage abstractions for AI agent systems. Implements a provider-agnostic interface for persisting agent state, conversation history, and knowledge artifacts across multiple storage backends.

## Tech Stack

- Language: Go
- Module: digital.vasic.storage

## Key Features

- Provider-agnostic storage interface
- Pluggable backend implementations (local, cloud, database)
- Structured and unstructured data persistence
