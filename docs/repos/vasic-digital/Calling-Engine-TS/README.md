- **Category**: TypeScript / React + Web
- **Status**: Active

## Overview

A TypeScript calling engine built on SIP.js and WebRTC for real-time voice and video communication. Provides a generic abstraction over SIP signaling and media handling for browser-based calling applications.

## Tech Stack

- Language: TypeScript
- Protocols: SIP.js, WebRTC
- Use case: Voice/video calling

## Key Features

- SIP.js-based SIP signaling
- WebRTC media stream management
- Generic calling engine abstraction
