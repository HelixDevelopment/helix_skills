# Auth-Context-React

- **GitHub URL**: <https://github.com/vasic-digital/Auth-Context-React>
- **Description**: React AuthProvider and useAuth hook for Catalogizer authentication with React Query
- **Category**: TypeScript / React + Web
- **Status**: Active

## Overview

A React authentication context providing an AuthProvider component and useAuth hook for the Catalogizer ecosystem. Integrates with React Query for token management, session handling, and authenticated API requests.

## Tech Stack

- Language: TypeScript
- Framework: React
- Data fetching: React Query

## Key Features

- AuthProvider context for React applications
- useAuth hook for accessing authentication state
- React Query integration for token management
