- **Category**: TypeScript / React + Web
- **Status**: Active

## Overview

React components for managing media collections within the Catalogizer application. Provides UI for creating, organizing, and browsing collections of media entities with drag-and-drop and bulk operations.

## Tech Stack

- Language: TypeScript
- Framework: React
- Target: Catalogizer

## Key Features

- Collection creation and organization UI
- Drag-and-drop collection management
- Bulk operations for collection items
