- **Category**: Apps / Tools / Experimental

## Overview

A cross-platform FFmpeg wrapper toolkit for mobile and desktop applications. Provides FFmpeg capabilities on Android, Flutter, iOS, Linux, macOS, React Native, and tvOS. Supersedes MobileFFmpeg, flutter_ffmpeg, and react-native-ffmpeg.

## Tech Stack

- Language: Multiple (C, Java, Swift, Dart, JavaScript)
- Core: FFmpeg
- Platforms: Android, Flutter, iOS, Linux, macOS, React Native, tvOS

## Key Features

- Cross-platform FFmpeg integration
- Media transcoding and processing
- Support for 7+ platforms from a single toolkit
