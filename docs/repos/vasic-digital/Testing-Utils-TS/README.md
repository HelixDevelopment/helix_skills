## Overview

A collection of generic testing utilities for React and Redux applications. Provides custom render helpers, mock providers, and test fixtures for writing reliable component and integration tests.

## Tech Stack

- Language: TypeScript
- Framework: React Testing Library
- Target: React + Redux applications

## Key Features

- Custom React render helpers with providers
- Redux store mock factories
- Test fixtures and assertion utilities
