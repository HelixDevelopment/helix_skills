# Document-KMP

- **GitHub URL**: <https://github.com/vasic-digital/Document-KMP>
- **Description**: Kotlin Multiplatform document model with format detection and change tracking
- **Category**: Cross-platform (KMP) Modules
- **Status**: Active

## Overview

Document-KMP provides a cross-platform document model for Kotlin Multiplatform applications. It includes format detection to identify document types from content, and change tracking to maintain a history of modifications. The module enables shared document handling logic across Android, iOS, desktop, and server targets.

## Tech Stack

- Language: Kotlin (KMP)
- Framework: Kotlin Multiplatform

## Key Features

- Platform-agnostic document model with shared data types
- Automatic format detection from document content
- Change tracking with modification history
- Cross-platform document lifecycle management
