## Overview

A reusable Go module for filesystem and resource watching. Provides event-driven monitoring of file changes, directory modifications, and resource state changes with configurable polling and notification mechanisms.

## Tech Stack

- Language: Go
- Module path: digital.vasic.watcher

## Key Features

- Filesystem change monitoring
- Event-driven file and directory watching
- Configurable polling and notifications
