## Overview

A reusable rate limiting module for Go applications. Provides token bucket, sliding window, and other rate limiting strategies for controlling request throughput in APIs and services.

## Tech Stack

- Language: Go
- Module path: digital.vasic.ratelimiter

## Key Features

- Token bucket and sliding window algorithms
- Configurable rate limiting strategies
- Thread-safe rate enforcement
