## Overview

An LLM benchmarking framework that runs standard evaluation suites including SWE-bench, HumanEval, and MMLU. Tracks results over time and produces leaderboards for comparing model performance across providers and versions.

## Tech Stack

- Language: Multiple
- Domain: LLM evaluation and benchmarking

## Key Features

- SWE-bench, HumanEval, and MMLU evaluation harnesses
- Automated leaderboard generation and historical tracking
- Cross-provider model comparison and reporting
