## Overview

A reusable Go module implementing retrieval-augmented generation pipelines. Provides document indexing, chunking, retrieval, and context assembly for grounding LLM responses in external knowledge sources.

## Tech Stack

- Language: Go
- Module: digital.vasic.rag

## Key Features

- Document chunking and indexing pipeline
- Semantic retrieval with configurable strategies
- Context assembly and ranking for LLM consumption
