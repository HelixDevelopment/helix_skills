# Formatters-KMP

- **GitHub URL**: <https://github.com/vasic-digital/Formatters-KMP>
- **Description**: Cross-platform KMP text format detection, parsing interfaces, and format registry
- **Category**: Cross-platform (KMP) Modules
- **Status**: Active

## Overview

Formatters-KMP provides cross-platform text format detection, parsing interfaces, and a format registry for Kotlin Multiplatform projects. It enables applications to identify text formats from content, register custom format parsers, and apply format-specific transformations. The registry pattern allows extensible format support across all KMP targets.

## Tech Stack

- Language: Kotlin (KMP)
- Framework: Kotlin Multiplatform

## Key Features

- Automatic text format detection from content inspection
- Extensible format registry for custom format parsers
- Platform-agnostic parsing interfaces
- Cross-platform text format transformation utilities
