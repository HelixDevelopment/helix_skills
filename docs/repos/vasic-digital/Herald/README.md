# Herald

- **GitHub URL**: <https://github.com/vasic-digital/Herald>

- **Category**: Misc / Scaffolds

## Overview

A notification fan-out system that ingests events from various sources and reliably distributes them to multiple notification channels. Ensures every alert reaches its intended destination through configurable routing rules, preventing notification confusion and ensuring delivery guarantees.

## Tech Stack

- Language: Multiple
- Category: Event notification and alerting
- Pattern: Fan-out / pub-sub

## Key Features

- Multi-channel notification fan-out
- Reliable event delivery guarantees
- Configurable alert routing rules
