Repository catalogue for the `HelixDevelopment` GitHub organisation. Browse individual repo pages for full details.

## Organisation Overview

HelixDevelopment is the AI and gaming platform organisation within the Helix ecosystem. It provides the core cognitive engine (HelixAgent), the QA framework (helixqa), the governance layer (HelixConstitution), LLM infrastructure (LLMProvider, HelixLLM, LLMOrchestrator), development tools (HelixCode, HelixBuilder, HelixSpecifier), and the gaming platform (HelixPlay, Helix-Game-1, Stake-Tetris).
