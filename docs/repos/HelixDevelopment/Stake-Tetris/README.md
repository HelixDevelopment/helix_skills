# Stake-Tetris

- **GitHub URL**: <https://github.com/HelixDevelopment/Stake-Tetris>
- **Description**: Tetris-style puzzle game with Stake integration -- classic gameplay combined with in-game economy, competitive leaderboards, and multiple game modes
- **Category**: Gaming
- **Status**: Active

## Overview

Stake-Tetris combines classic falling-block Tetris gameplay with Stake-based economy features for the Helix gaming platform. It provides an engaging casual gaming experience with in-game rewards, competitive leaderboards, and multiple game modes (classic, timed, challenge, multiplayer). Built on HelixPlay infrastructure, it demonstrates the platform's capability for casual competitive gaming.

## Tech Stack

- Language: TypeScript / JavaScript
- Rendering: Canvas/WebGL (web-based), HTML5 game patterns
- Architecture: Client-side game engine with server-side Stake validation
- Key patterns: Game loop, input buffering, animation tweening, state machine

## Key Features

- Classic Tetris gameplay with falling blocks, rotation, line clearing, and scoring
- Stake integration for in-game economy -- earn and spend through gameplay
- Competitive leaderboard system with real-time ranking updates
- Multiple game modes (classic, timed, challenge, multiplayer)
- Progressive difficulty scaling with configurable speed curves
