# Stake-Tetris

- **GitHub URL**: <https://github.com/HelixDevelopment/Stake-Tetris>
- **Description**: Stake Tetris -- a Tetris-style puzzle game with Stake integration for the Helix gaming platform. Combines classic falling-block gameplay with Stake-based economy features, providing an engaging casual gaming experience with in-game rewards and competitive leaderboards.
- **Category**: HelixDevelopment
- **Status**: Active

## Capabilities

- Classic Tetris gameplay with falling blocks, rotation, line clearing, and scoring
- Stake integration for in-game economy -- earn and spend through gameplay
- Competitive leaderboard system with real-time ranking updates
- Multiple game modes (classic, timed, challenge, multiplayer)
- Progressive difficulty scaling with configurable speed curves
- Visual effects and animations for line clears, combos, and achievements
- Sound system with music and responsive sound effects
- Save/resume game state with cloud persistence
- Configurable controls supporting keyboard, touch, and gamepad input

## Technology

- **Language**: TypeScript / JavaScript
- **Frameworks**: Web-based rendering (Canvas/WebGL), HTML5 game patterns
- **Architecture**: Client-side game engine with server-side Stake validation
- **Key patterns**: Game loop, input buffering, animation tweening, state machine

## Integration

- Built on HelixPlay gaming platform infrastructure for rendering and input
- Uses Helix-Game-Demo patterns as the development starting template
- Integrates with Stake systems for economy and reward transactions
- Connects to VisionEngine for automated visual testing of game screens
- Game analytics feed into the Helix project management system
- Shares game utilities and patterns with Helix-Game-1

## Status

Active development. Core Tetris gameplay and Stake integration functional. Multiple game modes implemented. Leaderboard system operational. Ongoing work on multiplayer mode and visual polish.
